Usage Rights

Depending on the license you purchase, you are granted the following rights:

  • Indie License: Use in one commercial or non-commercial project by a single developer or small team (up to 3 people).
  • Professional License: Use in up to five commercial or non-commercial projects by a team of any size.
  • Enterprise License: Use in unlimited commercial or non-commercial projects by a team of any size.

All licenses include the right to distribute the compiled version of your game without any additional royalties or fees.

Restrictions

The following restrictions apply to all license types:

  • You may not resell, redistribute, or sublicense the inventory system source code itself.
  • You may not use the inventory system in a game template that you sell to others.
  • You may not create derivative asset packs based on this inventory system.
  • You may not claim that you created the inventory system.

These restrictions are in place to protect our intellectual property while allowing you the freedom to use the system in your games.

Attribution

Attribution is appreciated but not required. If you wish to provide attribution, you can include the following in your game's credits:

"Inventory system powered by InventoryPro for Godot"

You may also include a link to our website if you wish, but this is entirely optional.

Modifications

You are permitted to modify the inventory system source code to suit your project needs. However:

  • Modifications do not grant you additional rights beyond what your license permits.
  • You may not distribute your modified version of the source code separately from your game.
  • You retain all modifications you make for your own use.

The Enterprise license includes support for custom modifications if you need assistance adapting the system to your specific requirements.

Support & Updates

Support and update policies vary by license:

  • Indie License: Basic email support and minor updates for 6 months.
  • Professional License: Priority email support and all updates for the lifetime of the product.
  • Enterprise License: Priority email and Discord support, all updates for the lifetime of the product, and custom modification assistance.

We are committed to maintaining compatibility with the latest versions of Godot and addressing any bugs or issues that arise.

Frequently Asked Questions

Find answers to common questions about our inventory system

Is the inventory system compatible with Godot 4.4?

Yes, our inventory system is fully compatible with Godot 4.4. We actively maintain the system and update it to support the latest Godot versions as they are released.

Can I use the inventory system in a commercial game?

Absolutely! All license types (Indie, Professional, and Enterprise) allow you to use the inventory system in commercial games. The only difference is the number of projects you can use it in, as specified in the license details.

Do I need programming skills to use this system?

Basic GDScript knowledge is recommended to integrate the inventory system into your game effectively. However, the system comes with comprehensive documentation and examples to help you get started, even with limited programming experience.

Can I modify the inventory system to fit my game's needs?

Yes, all licenses allow you to modify the source code to customize the inventory system for your specific requirements. The modular design makes it easy to adapt to different game types and UI styles.

What kind of support is provided?

Support varies by license type. The Indie license includes basic email support for 6 months. The Professional license offers priority email support for the lifetime of the product. The Enterprise license provides priority email and Discord support, plus assistance with custom modifications.

Can I see a demo before purchasing?

Yes, we offer a limited demo version that you can try before purchasing. The demo includes basic functionality but has some features disabled. Visit our website to download the demo and explore the system.

How does the save/load system work?

Our inventory system integrates seamlessly with Godot's built-in save/load functionality. It uses a serialization system to convert inventory data into a format that can be easily saved and loaded. The system handles complex item data, including custom properties, quantities, and equipped status.

Can I use this for mobile games?

Yes, the inventory system is fully compatible with mobile platforms. It includes touch-friendly interfaces and optimized performance for mobile devices. The UI automatically adapts to different screen sizes and orientations.
